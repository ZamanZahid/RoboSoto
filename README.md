# RoboSoto

![RoboSoto](Robot%20Images/Final-RoboSoto.png)

## Overview
RoboSoto is a robot car I made. The robot uses a Romeo ESP32-S3 controller and has remote-controlled navigation.

## Features
- Wireless ESP-NOW joystick control
- Four-wheel differential drive
- Collision detection with bumper switches
- Autonomous obstacle avoidance
- Photoresistor light tracking
- BNO055 IMU navigation
- IMU-based square and figure-8 driving

## Hardware
- Romeo ESP32-S3
- FireBeetle ESP32
- BNO055 IMU
- Photoresistor
- 4 DC motors
- Motor driver
- Joystick controller
- Bumper switches
- Battery pack

## Software
- Arduino IDE
- ESP-NOW communication
- Adafruit BNO055 library
